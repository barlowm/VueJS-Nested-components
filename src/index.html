Nested Component Data Passing

Sample VueJS app to pass simple data from a component nested several levels deep

Start at the lowest level (credential_input.js) with a custom input field component which accepts input from the user and passes it up to it's parent

The next level up (credentials.js) has several of the custom input field components that receive data from each sub component and combines them into a single object which is then passed up

At the top component level (app.js) we have a single object which receives the single component object from it's child, which has all the credential values collected in the "credentials" component

Note: This app uses the standalone "Electron" vue-devtools app for debugging
